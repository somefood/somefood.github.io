마크다운(markdown) 정리
================

##### github 블로그를 다시 정리하려니, 요 마크다운의 문법을 알아두는게 꽤 필요하기에 정리하기로 하였다.

# 문법 정리

## 1. 헤더(Headers)
- 큰 제목: 문서 제목으로 사용
```
큰 제목
======
```
ex) 큰 제목
======

- 작은 제목: 부제목으로 사용
```
작은 제목
------
```

ex) 작은 제목
-------

- 글머리: 1~6까지 표현 가능
```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

# H1
## H2
### H3
#### H4
##### H5
###### H6

## 2. BlockQuote
```
> 첫 번째 블록쿼트
>> 두 번째 블록쿼트
>>> 세 번째 블록쿼트
```
> 첫 번째 블록쿼트
>> 두 번째 블록쿼트
>>> 세 번째 블록쿼트

## 3. 목록
#### 순서있는 목록
```
1. 일!
2. 이!
3. 삼!
```
1. 일!
2. 이!
3. 삼!

#### 순서없는 목록
```
- 가
- 나
- 다
```
- 가
- 나
- 다

## 4. 링크
```
[깃허브](https://somefood.github.io)
```
[깃허브](https://somefood.github.io)

## 5. 강조
```
기울여쓰기(italic) *글자* or _글자_
굵게쓰기(bold) **글자** or __글자__
```

*이태릭* _이태릭_

**볼드** __볼드__


## 6. 표
```
|칼럼1|칼럼2|
|--|--|
|내용1|내용2|
|내용3|내용4|
```

|칼럼1|칼럼2|
|--|--|
|내용1|내용2|
|내용3|내용4|
